# 三套独立模板库 - 增强功能说明

## 📋 功能概述

为后台管理系统深化设计了**三套独立的模板库**：
1. **任务模板库** - 培养孩子的责任感、自理能力和品格
2. **心愿模板库** - 激励孩子的奖励选项
3. **习惯模板库** - 帮助养成良好生活习惯

每套模板库都包含：
- ✅ 深入分类（主分类 + 子分类）
- ✅ 精美配图（Unsplash高质量图片）
- ✅ 详细文字介绍
- ✅ 使用建议和实施方案
- ✅ 适用年龄、难度等级、标签等元数据

---

## 🗄️ 数据库迁移

### 1. 执行增强迁移脚本

```bash
# 在 Supabase SQL Editor 中执行
supabase-templates-enhanced-migration.sql
```

**新增字段：**
- `subcategory` - 子分类
- `image_url` - 配图URL
- `detailed_description` - 详细介绍
- `usage_suggestions` - 使用建议
- `age_range` - 适用年龄
- `difficulty` - 难度等级（easy/medium/hard）
- `tags` - 标签数组
- `estimated_time` - 预估时间
- `materials_needed` - 所需材料
- `learning_outcomes` - 学习成果
- `is_featured` - 是否精选
- `sort_order` - 排序顺序

**新增表：**
- `template_categories` - 模板分类表（支持多级分类）
- `template_usage_stats` - 模板使用统计表

## 🎨 后台管理界面

### 3. 访问增强版模板管理

启动后台管理系统：

```bash
cd admin
npm run dev
```

访问：`http://localhost:5174/templates`

**新功能特性：**

#### 📊 三库独立展示
- 顶部Tab切换：任务模板 | 习惯模板 | 心愿模板
- 每个Tab显示对应模板数量和统计

#### 🎯 视觉化展示
- **网格视图**：卡片式布局，显示配图、标题、星星数、分类标签
- **列表视图**：表格布局，适合快速浏览和比较

#### 🔍 丰富的信息展示
每张卡片显示：
- 高清配图（从Unsplash获取）
- 标题和简介
- 分类 + 子分类
- 难度等级（简单/中等/困难，用颜色区分）
- 适用年龄范围
- 标签（如"整理"、"责任感"、"创造力"等）
- 使用次数统计
- 精选标记（⭐）

#### 📝 详情弹窗
点击任意模板卡片，弹出详情窗口，显示：
- 大图展示
- 完整简介
- **详细介绍**（实施方案、步骤、注意事项）
- **使用建议**（家长指导建议）
- 标签云
- 统计数据（使用次数、来源、状态）

#### 🛠️ 管理功能
- **启用/停用**：快速切换模板状态
- **设为精选**：标记优质模板，在前端优先展示
- **删除模板**：移除不需要的模板
- **搜索过滤**：支持按标题、描述、标签搜索

## 🎯 模板内容示例

### 任务模板示例："洗衣服"
```yaml
标题: 洗衣服
分类: 家务责任 > 衣物整理
难度: 中等 (8-14岁)
星星: 3⭐
配图: [洗衣机图片]

简介:
  在家长监督下，学习操作洗衣机洗涤自己的小件衣物（如袜子、内裤、手帕）。
  了解洗衣程序、用量和分类知识。

详细介绍:
  从最简单的料理开始（如煮鸡蛋、做三明治、拌沙拉），逐步学习烹饪技能。
  培养孩子的食品安全意识和营养知识。

使用建议:
  首次操作需要家长全程陪同，讲解每个步骤的原理（为什么深浅色要分开、
  柔顺剂的作用等）。可以制作"洗衣指南卡"供孩子参考。

标签: 生活技能、营养、创造力
适用年龄: 8-14岁
所需时间: 30-60分钟
所需材料: 洗衣机、洗衣液、洗衣指南卡
学习成果: 生活技能、营养知识、创造力、安全意识
```

## 🐛 常见问题

### Q1: 执行迁移脚本时报错 "column already exists"
**A:** 迁移脚本使用了 `IF NOT EXISTS` 和 `IF EXISTS`，应该不会报错。如果报错，可以手动注释掉已存在的列。

### Q2: 图片无法显示
**A:** 当前使用Unsplash外部链接。如果无法访问，可以：
1. 使用VPN
2. 将图片下载后上传到Supabase Storage
3. 修改 `image_url` 字段为Storage的路径

### Q3: 如何添加自定义模板？
**A:** 当前版本暂未实现"添加模板"功能，可以通过以下方式添加：
1. 直接在Supabase数据库中插入记录
2. 使用SQL脚本批量导入
3. 等待后续版本支持界面操作

### Q4: 种子数据可以修改吗？
**A:** 可以！在后台管理界面点击模板卡片，查看详情。虽然当前版本不支持编辑，但可以直接在Supabase中修改。
