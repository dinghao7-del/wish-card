# Stitch Prompt - 切换用户界面

## 页面基本信息
- 页面名称：切换用户（Switch User）
- 页面类型：用户选择 / 模式切换
- 优先级：P0（核心交互）
- 设计迭代：预计2次

## 交互动画

### 切换动画序列

```
帧 1：点击「切换为小明」
        ↓
帧 2：当前用户卡片缩小 + 淡出
        ↓
帧 3：选中用户卡片放大 + 淡入
        ↓
帧 4：显示「切换成功」Toast
        ↓
帧 5：自动关闭页面或返回首页
```

### 卡片点击效果
- 点击时：缩放 0.95
- 选中时：边框颜色变化动画
- 选中标记：从小圆点放大出现
