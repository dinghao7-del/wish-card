## 二、AI 语音助手详细设计

### 2.1 语音技术方案

| 层级 | 技术选型 | 说明 |
|------|---------|------|
| **语音识别（STT）** | Web Speech API (`SpeechRecognition`) | 浏览器原生、零成本、中文识别率高；不支持的浏览器自动降级为文字输入 |
| **语音合成（TTS）** | Web SpeechSynthesis API | 浏览器原生、免费播报反馈；可切换中文女声/男声 |
| **意图理解（NLU）** | Gemini 1.5 Flash | 基于现有 `@google/genai` 扩展，支持 40+ 意图识别 |
| **执行引擎** | VoiceCommandExecutor | 对接 `FamilyContext`，执行实际业务操作 |
| **UI 层** | 全局浮动语音按钮 + 对话面板 + 快捷指令 | 右下角入口，对话式交互 |

### 2.3 日程四象限分析详解

这是语音助手的核心差异化功能。基于 **艾森豪威尔矩阵（紧急-重要四象限）** 对用户当天任务进行智能分类：

```
                    紧急                    不紧急
              ┌─────────────────┬─────────────────┐
   重要       │  Q1 立即执行     │  Q2 计划安排     │
              │  (今天必做任务)   │  (重要不急任务)   │
              ├─────────────────┼─────────────────┤
   不重要     │  Q3 委托/简化    │  Q4 适当放弃     │
              │  (紧急杂事)      │  (低价值活动)     │
              └─────────────────┴─────────────────┘
```

**分类规则（AI 自动推断）：**

| 象限 | 分类依据 | 示例 |
|------|---------|------|
| **Q1 紧急+重要** | 今天截止 + 高星星奖励(≥30) + 家长指定 | "做作业"(截止今天, 50⭐), "练字"(待审核) |
| **Q2 重要不紧急** | 习惯类任务 + 长期目标 + 中等星星 | "每日早起"(习惯, 20⭐), "阅读30分钟"(40⭐) |
| **Q3 紧急不重要** | 低星星日常 + 被分配但非自选 | "收拾书包"(10⭐), "刷牙洗脸"(10⭐) |
| **Q4 不紧急不重要** | 惩罚类习惯 + 零星杂项 | "晚睡"(-20⭐), "挑食"(-10⭐) |

**AI Prompt 模板：**

```
你是一个日程分析助手。根据以下任务列表，按照艾森豪威尔四象限法则进行分类。
```

### 2.4 VoiceCommandExecutor 架构

```typescript
// src/lib/voiceCommandExecutor.ts

interface VoiceCommand {
  intent: string;          // 意图标识，如 'task.create'
  params: Record<string, any>; // AI 提取的参数
  confidence: number;      // 置信度 0-1
}

class VoiceCommandExecutor {
  private familyContext: FamilyContextType;
  private navigate: (path: string) => void;

  async execute(command: VoiceCommand): Promise<VoiceResult> {
    switch (command.intent) {
      case 'task.create':     return this.createTask(command.params);
      case 'task.edit':       return this.editTask(command.params);
      case 'task.delete':     return this.deleteTask(command.params);
      case 'task.complete':   return this.completeTask(command.params);
      case 'task.approve':    return this.approveTask(command.params);
      case 'habit.create':    return this.createHabit(command.params);
      case 'habit.checkin':   return this.checkinHabit(command.params);
      case 'reward.create':   return this.createReward(command.params);
      case 'reward.redeem':   return this.redeemReward(command.params);
      case 'stars.check':     return this.checkStars(command.params);
      case 'stars.add':       return this.addStars(command.params);
      case 'schedule.quadrant': return this.analyzeQuadrant(command.params);
      case 'schedule.suggest':  return this.suggestSchedule(command.params);
      case 'pomodoro.start':  return this.startPomodoro(command.params);
      case 'navigate':        return this.navigateTo(command.params);
      case 'member.switch':   return this.switchMember(command.params);
      case 'chat':            return this.generalChat(command.params);
      // ... 其他 40+ 意图
    }
  }
}
```

### 2.7 语音交互流程

```
用户按住 Mic / 输入文字
        ↓
  SpeechRecognition 识别 / 直接获取文字
        ↓
  发送到 Gemini AI（携带完整上下文）
        ↓
  AI 返回 { intent, params, reply, needConfirm }
        ↓
  ┌─ needConfirm=true  → 展示确认对话框 → 用户确认/修改
  │                                           ↓
  └─ needConfirm=false → 直接执行
        ↓
  VoiceCommandExecutor.execute(command)
        ↓
  调用 FamilyContext 对应方法
        ↓
  更新 UI + 语音播报结果（TTS）
```

### 3.2 .ics 日历订阅方案（通用方案）

```
愿望卡 App（Supabase）
        ↓ 生成 .ics 文件
```


#### 3.2.3 Supabase Edge Function（生成 .ics）

```typescript
// supabase/functions/calendar-ics/index.ts

import { serve } from 'https://deno.land/std@0.168.0/http/server.ts'
import { createClient } from 'https://esm.sh/@supabase/supabase-js@2'

serve(async (req) => {
  const url = new URL(req.url)
  const familyId = url.searchParams.get('family')
  const token = url.searchParams.get('token')

  // 验证 token（防止未授权访问）
  const supabase = createClient(Deno.env.get('SUPABASE_URL')!, Deno.env.get('SUPABASE_SERVICE_ROLE_KEY')!)
  const { data: family } = await supabase.from('families').select('*').eq('id', familyId).single()
  if (!family) return new Response('Not found', { status: 404 })

  // 查询家庭的所有任务
  const { data: tasks } = await supabase
    .from('tasks')
    .select('*')
    .eq('family_id', familyId)
    .eq('completed', false)

  const { data: members } = await supabase
    .from('members')
    .select('*')
    .eq('family_id', familyId)
    .eq('is_active', true)

  // 生成 .ics 内容
  let ics = 'BEGIN:VCALENDAR\nVERSION:2.0\nPRODID:-//愿望卡//Forest Family//CN\n'
  ics += 'CALSCALE:GREGORIAN\nMETHOD:PUBLISH\n'
  ics += 'X-WR-CALNAME:愿望卡-家庭日程\nX-WR-TIMEZONE:Asia/Shanghai\n'

  for (const task of tasks || []) {
    const assigneeNames = (task.assignee_ids || [])
      .map((id: string) => members?.find((m: any) => m.id === id)?.name || '')
      .filter(Boolean).join('、')

    ics += 'BEGIN:VEVENT\n'
    ics += `DTSTART;TZID=Asia/Shanghai:${formatDate(task.created_at)}T090000\n`
    ics += `DTEND;TZID=Asia/Shanghai:${formatDate(task.created_at)}T100000\n`
    ics += `SUMMARY:${task.title} ⭐${task.star_amount}\n`
    ics += `DESCRIPTION:${task.description || ''}\\n奖励：${task.star_amount}颗星星\\n指派给：${assigneeNames}\n`
    ics += `UID:task-${task.id}@forest-family\n`
    ics += 'STATUS:CONFIRMED\n'
    ics += 'BEGIN:VALARM\nTRIGGER:-PT30M\nACTION:DISPLAY\n'
    ics += `DESCRIPTION:任务即将开始：${task.title}\n`
    ics += 'END:VALARM\n'
    ics += 'END:VEVENT\n'
  }

  ics += 'END:VCALENDAR'

  return new Response(ics, {
    headers: {
      'Content-Type': 'text/calendar; charset=utf-8',
      'Content-Disposition': 'attachment; filename="forest-family.ics"',
      'Cache-Control': 'no-cache, max-age=300',
    },
  })
})

function formatDate(isoString: string): string {
  const d = new Date(isoString)
  return `${d.getFullYear()}${String(d.getMonth()+1).padStart(2,'0')}${String(d.getDate()).padStart(2,'0')}`
}
```

```typescript
// supabase/functions/google-calendar-sync/index.ts

// 推送任务到 Google Calendar
async function pushToGoogleCalendar(task: any, token: CalendarToken) {
  const event = {
    summary: `${task.title} ⭐${task.star_amount}`,
    description: `${task.description}\n奖励：${task.star_amount}颗星星`,
    start: { dateTime: task.startTime, timeZone: 'Asia/Shanghai' },
    end: { dateTime: calculateEndTime(task.startTime), timeZone: 'Asia/Shanghai' },
    reminders: { useDefault: false, overrides: [
      { method: 'popup', minutes: 30 },
      { method: 'popup', minutes: 10 },
    ]},
    colorId: task.isHabit ? '2' : '9', // 绿色=习惯，蓝色=任务
  }

  const response = await fetch(
    `https://www.googleapis.com/calendar/v3/calendars/${token.calendar_id}/events`,
    {
      method: 'POST',
      headers: { Authorization: `Bearer ${token.access_token}` },
      body: JSON.stringify(event),
    }
  )
  return response.json()
}

// 从 Google Calendar 拉取事件（双向同步）
async function pullFromGoogleCalendar(token: CalendarToken, since: string) {
  const response = await fetch(
    `https://www.googleapis.com/calendar/v3/calendars/${token.calendar_id}/events?` +
    `updatedMin=${since}&singleEvents=true&orderBy=startTime`,
    { headers: { Authorization: `Bearer ${token.access_token}` } }
  )
  const data = await response.json()
  return data.items || []
}
```
