```markdown
├────────────────────────────────────────────────────────────┤
│ 星星动画                                                          │
│ duration: 600ms         easing: spring                            │
│ 用途：完成任务获得星星、兑换成功                                    │
│ 效果：星星从任务图标位置飞向余额区域                                │
└────────────────────────────────────────────────────────────┘

## 特殊动画效果

### 打卡成功动画
1. 星星图标从任务位置缩小并向上飞出
2. 留下任务名称文字（淡出）
3. 多个小星星从中心向外扩散
4. 显示「+5 ⭐」文字飘向余额区
5. 余额数字跳动动画

### 兑换成功动画
1. 屏幕出现礼花/撒花效果
2. 星星聚拢动画
3. 奖励图标放大出现
4. 显示「兑换成功」文字
5. 3秒后自动消失

### 下拉刷新动画
1. 蓝色圆圈 + 星星图标
2. 下拉时星星旋转
3. 释放时星星向上飞出
4. 显示「刷新成功」
```
