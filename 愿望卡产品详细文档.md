# 愿望卡 (WishCard) - 产品详细文档

**文档版本**: v1.0  
**更新日期**: 2026年4月29日  
**产品类型**: 家庭习惯养成/奖励管理应用

---

## 一、产品概述

### 1.1 产品定位
愿望卡是一款面向中国家庭设计的习惯养成与奖励管理应用，通过"星星积分"激励机制，帮助家长培养孩子的好习惯，实现家庭成员间的协作与互动。

### 1.2 核心价值
- **激励驱动**: 用星星积分量化努力，建立正向激励闭环
- **家庭协作**: 多成员参与，家长监管，孩子执行
- **习惯追踪**: 支持任务打卡、习惯养成、奖惩记录
- **数据可视化**: 排行榜、四象限分析、日历同步

### 1.3 目标用户
| 用户角色 | 核心需求 |
|---------|---------|
| 家长 | 创建任务/习惯、审批打卡、发放奖励、管理成员 |
| 孩子 | 执行任务、积累星星、兑换心愿 |
| 家庭管理员 | 数据备份、成员管理、权限控制 |

## 二、技术架构

### 2.1 技术栈
```
前端框架: React 18 + TypeScript + Vite
路由管理: React Router v6
UI框架: Tailwind CSS + Material Design 3
动画效果: Framer Motion
后端服务: Supabase (PostgreSQL + Auth + Realtime)
移动端: Capacitor (iOS/Android PWA)
图标库: Lucide React
国际化: i18next (6种语言支持)
```
