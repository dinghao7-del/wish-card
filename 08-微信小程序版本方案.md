# Forest Family - 微信小程序版本方案

> 基于 Stitch 设计稿，面向微信生态的完整开发方案

---

## 一、技术选型对比

### Flutter App vs 微信小程序

| 维度 | Flutter App | 微信小程序 |
|:-----|:------------|:----------|
| **分发方式** | 应用商店下载 | 扫码/搜索即用 |
| **安装门槛** | 需要下载安装 | 无需安装 |
| **用户体量** | 需要推广 | 微信13亿用户 |
| **登录方式** | 自建账号体系 | 微信一键授权 |
| **家庭共享** | 需要邀请机制 | 微信好友天然打通 |
| **消息通知** | App推送 | 微信订阅消息 |
| **开发成本** | 较高 | 较低 |
| **审核周期** | 1-7天 | 1-7天 |

**结论：微信小程序是 Forest Family 更合适的形态** ✅
- 家庭成员都在用微信，无需额外安装
- 任务提醒可用微信订阅消息，打开率远高于App推送
- 微信登录免去账号注册门槛

---

## 二、推荐技术栈

### 方案一：原生小程序 + 云开发（推荐 ⭐⭐⭐⭐⭐）

```
前端：原生 WXML + WXSS + JS
后端：微信云开发（零服务器成本）
数据库：云数据库（MongoDB）
存储：云存储（图片/音频）
函数：云函数（业务逻辑）
```

**优势：**
- 无服务器费用（小程序免费额度足够早期使用）
- 微信登录、微信支付、订阅消息开箱即用
- 云开发控制台直接管理数据

---

### 方案二：Taro 框架（推荐 ⭐⭐⭐⭐）

```
前端：React + Taro
后端：微信云开发
状态管理：Zustand / Redux
```

**优势：**
- 一套代码，可同时编译为小程序 + H5 + App
- React 生态丰富，组件库多
- 未来扩展到其他平台成本低

---

### 我的推荐

**用原生小程序 + 云开发**，原因：
1. 直接对应 Stitch 设计稿，无需框架转换
2. AI Studio 生成小程序代码准确度更高
3. 云开发免费额度够用，无额外成本
4. 调试最简单，问题最少

## 五、核心功能实现

### 1. 微信登录 + 家庭绑定

```javascript
// utils/auth.js
async function login() {
  // 1. 获取微信登录凭证
  const { code } = await wx.login()
  
  // 2. 调用云函数换取 openid
  const { openid } = await wx.cloud.callFunction({
    name: 'login',
    data: { code }
  })
  
  // 3. 查询用户是否已有家庭
  const db = wx.cloud.database()
  const user = await db.collection('users').where({ openid }).get()
  
  if (user.data.length === 0) {
    // 新用户 → 进入创建/加入家庭页面
    wx.navigateTo({ url: '/pages/onboarding/index' })
  } else {
    // 老用户 → 进入首页
    wx.switchTab({ url: '/pages/index/index' })
  }
}
```

### 2. 任务打卡 + 星星奖励

```javascript
// pages/check-in/index.js
Page({
  async submitCheckIn() {
    const { taskId, type, content } = this.data
    
    // 1. 上传打卡凭证（如果是图片）
    let cloudPath = ''
    if (type === 'photo') {
      cloudPath = await uploadToCloud(content)
    }
    
    // 2. 创建打卡记录
    const db = wx.cloud.database()
    await db.collection('checkIns').add({
      data: {
        taskId,
        userId: getApp().globalData.openid,
        type,
        content: cloudPath || content,
        starsEarned: this.data.task.rewardStars,
        createdAt: db.serverDate()
      }
    })
    
    // 3. 更新用户星星余额
    await db.collection('users').doc(userId).update({
      data: {
        starBalance: db.command.inc(this.data.task.rewardStars)
      }
    })
    
    // 4. 发送订阅消息通知家长
    await wx.cloud.callFunction({
      name: 'sendNotify',
      data: {
        toUserId: parentId,
        taskName: this.data.task.title,
        childName: childName
      }
    })
    
    // 5. 显示成功动画
    wx.navigateTo({ url: '/pages/check-in-success/index' })
  }
})
```
