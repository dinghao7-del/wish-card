## 一、产品概述

### 1.1 产品定位

**比乐时** 是一款由深圳市福驾科技有限公司开发的**家庭协作日程管理工具**，核心理念为「比肩同乐，时岁共珍」，专注提升家庭协作效率，帮助家庭成员实现高效时间管理。

### 1.2 目标用户

| 用户群体 | 特征描述 |
|---------|---------|
| **核心用户** | 6-12岁儿童的家长，尤其"老母亲"焦虑型家长 |
| **使用场景** | 解决"孩子作业拖拉+家务不主动"的家庭痛点 |
| **家庭结构** | 多成员家庭，需要协同管理日程和任务 |

| **跨天事项显示** | 优化跨越午夜的任务显示逻辑 | P1 |
| **日历起始日设置** | 支持自定义周一或周日作为一周起始 | P1 |

1. **数据结构**：`CalendarEvent { id, title, description, start_time, end_time, all_day, recurrence_rule, member_ids, color }`

3. **视图切换**：使用 ViewPager2 或类似组件实现流畅切换

## 四、数据库设计

### 4.1 数据表设计

#### 4.1.1 用户与认证

```sql
-- 用户表
CREATE TABLE users (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    email VARCHAR(255) UNIQUE NOT NULL,
    phone VARCHAR(20) UNIQUE,
    password_hash VARCHAR(255) NOT NULL,
    nickname VARCHAR(100),
    avatar_url TEXT,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- 家庭表
CREATE TABLE families (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    name VARCHAR(100) NOT NULL,
    invite_code VARCHAR(20) UNIQUE NOT NULL,
    created_by UUID REFERENCES users(id),
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- 家庭成员表
CREATE TABLE family_members (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    family_id UUID REFERENCES families(id) ON DELETE CASCADE,
    user_id UUID REFERENCES users(id) ON DELETE CASCADE,
    role VARCHAR(20) DEFAULT 'member',
    nickname VARCHAR(100),
    color VARCHAR(10) DEFAULT '#4A90D9',
    star_balance INTEGER DEFAULT 0,
    joined_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    UNIQUE(family_id, user_id)
);
```
